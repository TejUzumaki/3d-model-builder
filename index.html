3D Builder Studio Features

Asset Editor: Build 3D models using primitives and 2D planes. Use the Part Label field to name limbs (Head, LeftArm) so your AI can animate them automatically!

Terrain Editor: Generates a 3D volumetric terrain for game collisions. Use the Sculpt Tool to paint grooves, or use the Flatten brush to create flat areas for villages.

Asset Importer: In Terrain mode, use the dropdown to spawn your saved Asset projects directly onto the map.

Industry Tools: Use the View Presets for orthographic alignment, Snap to Grid for modular building, and Hide/Isolate to work on overlapping parts.

Controls & Shortcuts

  • G: Move | R: Rotate | S: Scale
  • F: Focus Camera | Del: Delete Object
  • Ctrl+Z: Undo | Ctrl+Shift+Z: Redo
  • Ctrl+D: Duplicate | Ctrl+G: Group
